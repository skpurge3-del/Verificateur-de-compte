## 📋 Description

**Vérificateur de Compte** est un outil de sécurité complet qui analyse la solidité d'un compte utilisateur. Il évalue plusieurs facteurs pour déterminer si un compte est bien protégé contre les menaces potentielles.

---

## 🎯 Fonctionnalités

| Fonctionnalité | Description |
|----------------|-------------|
| 🔑 **Analyse du mot de passe** | Vérifie la longueur, la complexité et les motifs courants |
| 🌍 **Support international** | Plus de **190 pays** avec leurs indicatifs téléphoniques |
| 📱 **Validation téléphone** | Accepte tous les formats de numéros du monde |
| 🛡️ **Authentification 2FA** | Évalue la méthode d'authentification secondaire |
| ⚠️ **Activité suspecte** | Prend en compte l'historique récent |
| 📊 **Score global** | Note sur 100 avec barre de progression |
| 💡 **Recommandations** | Conseils personnalisés selon le niveau |

---

## 🔍 Critères d'évaluation

### Mot de passe (50% du score)
- ✅ Minimum 12 caractères
- ✅ Au moins une majuscule
- ✅ Au moins une minuscule
- ✅ Au moins un chiffre
- ✅ Au moins un caractère spécial
- ✅ Pas d'information personnelle
- ❌ Pénalité pour motifs communs

### Authentification (30% du score)
| Méthode | Score |
|---------|-------|
| Aucune | 0% |
| SMS | 40% |
| Application | 70% |
| Biométrique | 80% |
| Clé physique | 90% |

### Activité récente (20% du score)
| Statut | Score |
|--------|-------|
| Aucune activité suspecte | 100% |
| Ce mois-ci | 70% |
| Cette semaine | 40% |
| Aujourd'hui | 10% |

---

## 📊 Niveaux de sécurité

| Niveau | Score | Icône | Signification |
|--------|-------|-------|---------------|
| **Solide** | 80-100% | 🛡️ | Compte très bien protégé |
| **Moyen** | 50-79% | ⚠️ | Protection à améliorer |
| **Faible** | 0-49% | 🚨 | Compte vulnérable |
